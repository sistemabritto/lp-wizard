# SKILL — Copy de Alta Conversão
# Protocolo S1 — Uso interno dos agentes de copy

## OS 3 CÉREBROS (ordem obrigatória: reptiliano → límbico → neocórtex)

| Cérebro | Ativado por | Função na copy |
|---|---|---|
| **Reptiliano** | Dor, medo, escassez, urgência, contraste brutal | Decide se fica lendo — milissegundos |
| **Límbico** | Histórias, identidade, raiva, esperança, pertencimento | Cria o desejo — segundos |
| **Neocórtex** | Dados, provas, garantias, lógica | Justifica a decisão já tomada |

**Erro fatal:** ativar neocórtex primeiro. Ninguém compra por lógica — compra por emoção e justifica com lógica.

---

## ERROS QUE MATAM CONVERSÃO

1. **Revelar demais** — curiosidade não satisfeita = compra. Explicação completa = sem razão para comprar.
2. **Culpar o avatar** — o problema é sempre do inimigo externo, nunca do leitor.
3. **Promessa genérica** — "LCP de 2.3s para 522ms" converte. "Página rápida" não converte.
4. **Headline fraca** — 80% decide em 3 segundos. Headline fraca = fim.
5. **Sem inimigo comum** — sem vilão não há tensão dramática.
6. **Sem garantia** — reptiliano interpreta compra sem garantia como ameaça.
7. **CTA único** — mínimo 3: hero (urgência), meio (lógica), final (emoção).

## REGRAS DE ESCRITA

**Headline:** promessa OU dor — não as duas. Máx 12 palavras. Nunca revele o mecanismo.
**Dores:** situação específica + consequência vivida. Nunca genérico, nunca culpa do leitor.
**Inimigo:** externo e específico. Nunca concorrente pelo nome. Nunca o próprio leitor.
**Custo da inação:** temporal e visual. Amplifique o real — não invente.
**CTA:** primeira pessoa do comprador ("Quero..."). Nunca "Clique aqui" ou "Comprar".
**Provocante:** não entregue — apenas o suficiente para criar desejo.
**Misteriosa:** específico pra ser crível, vago pra criar necessidade. Nunca entregue o segredo.

## PROMPTS POR SEÇÃO

```
[HEADLINE] produto={nome_produto} público={publico_comprador} dor={demanda_urgente} estado={estado_emocional}
→ 5 headlines que ativam reptiliano em 3s. Específicas, sem revelar mecanismo.

[DORES] produto={nome_produto} público={publico_comprador}
→ 3 dores: situação + consequência + emoção. Linguagem do avatar. Sem culpar o leitor.

[INIMIGO] dor={demanda_urgente} público={publico_comprador} produto={nome_produto}
→ Inimigo externo: quem é, como opera, por que o leitor não via. Sem nomear concorrentes.

[CUSTO INAÇÃO] público={publico_comprador} dor={demanda_urgente}
→ Cenário em 6 meses sem resolver. Visual, temporal, medo da perda. Só amplifique o real.

[3 PARTES] produto={nome_produto} prova={prova_resultado}
→ PROVOCANTE: "como assim isso existe?" sem revelar.
→ RACIONAL: dado mais específico e verificável disponível.
→ MISTERIOSA: só quem compra descobre — específico pra crer, vago pra desejar.
```
