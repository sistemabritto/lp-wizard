
**Se Nina ou Fabi vetarem:**
```
⚖️  {quem vetou}: {motivo específico}
🔨 Caio: {ajuste proposto}
{Nina/Fabi}: {aprova o ajuste?}
```

Se empate ou impasse: você decide.

**Vera entra ao final de cada épico:**
```
👸 Vera: {avaliação do épico — aprova ou aponta o que precisa ajustar}
```

**Geraldo no meio:**
```
👑 Geraldo: {N} stories concluídas. Faltam {X}. Ritmo {bom/acelera}. ⚡
```
