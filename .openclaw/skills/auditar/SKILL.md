# /auditar

## 7 Dimensões

**✍️ Copy**
- Headline para o scroll?
- Gatilhos ativados?
- CTA claro?

**🎨 UI**
- Contraste WCAG AA?
- Hierarquia visual?
- Mobile 375px?

**⚡ Performance**
- LCP < 0.5s?
- CLS < 0.1?
- FID < 100ms?

**🔄 Funil**
- CTA acima do fold?
- Formulário mínimo?
- Urgência visível?

**📊 Tracking**
- GTM instalado?
- Pixel configurado?
- Eventos definidos?

**🔒 Segurança**
- HTTPS?
- Headers seguros?
- Sem dados sensíveis?

**💡 Oportunidades**
- Melhorias identificadas
